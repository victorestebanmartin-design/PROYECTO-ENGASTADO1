Sistema de Engastado Automático

Guía rápida y demostración para operarios y administradores

Objetivo

Guiar y controlar el proceso de engastado de terminales, reduciendo errores y organizando el trabajo por paquetes.

  • Modo interactivo para operarios (V2)
  • Panel de administración para gestionar terminales y Excel
Presentación generada desde el proyecto local

Resumen corto

Una aplicación que lee un archivo Excel con terminales, muestra tarjetas por terminal y guía paso a paso la preparación de paquetes.

Para quién

  • Operarios de taller
  • Responsables de producción
  • Administradores

Demo - Flujo de uso (pasos)

  1. Abrir `http://localhost:5000/v2`
  2. Seleccionar la tarjeta del terminal
  3. Seguir instrucciones y temporizador
  4. Presionar Enter para avanzar o esperar el temporizador
  5. Reanudar o cambiar terminales según sea necesario

Pantalla de Administración

Subida y asociación de archivos Excel, badges con info (archivos procesados / con error / total) y grid de terminales con botones para activar/desactivar.

Inserta aquí capturas de `/admin` si quieres mostrar ejemplos visuales.

Cómo arrancar (Windows)

powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python run.py

Abrir en el navegador: / V2: http://localhost:5000/v2

Archivos importantes

  • run.py — iniciar servidor
  • app/excel_manager.py — leer Excel
  • static/js/main-v2.js — flujo operario
  • static/js/admin.js — panel `/admin`
  • static/css/style.css — estilos

FAQ rápido

¿Se pierde el progreso? — Actualmente se guarda en memoria; se puede añadir persistencia en base de datos.

¿Excel no aparece? — Subir desde `/admin` y asociar el código.
